# Document Q&A — Local LLM Integration

A full-stack web application that lets users paste any document and ask natural language questions about it. Answers are generated by a locally-hosted LLM with no data leaving the machine — built with public sector use cases in mind.

## Why this matters for government use cases
Government clients often cannot send sensitive documents to cloud APIs. This app runs entirely on local infrastructure — relevant to air-gapped and data-sensitive environments.

## Tech Stack
- **Frontend:** Next.js, TypeScript, Tailwind CSS
- **Backend:** Node.js (Next.js API routes)
- **AI:** Qwen3 8B running locally via Ollama
- **No external AI API calls** — fully self-hosted

## How it works
1. User pastes a document (report, policy, brief, etc.)
2. User types a question about the document
3. The app sends both to a local Qwen3 8B model via Ollama
4. The model answers based only on the provided document

## Running locally
1. Install [Ollama](https://ollama.com) and pull the model:
```bash
   ollama pull qwen3:8b
```
2. Install dependencies:
```bash
   npm install
```
3. Start the app:
```bash
   npm run dev
```
4. Open [http://localhost:3000](http://localhost:3000)
